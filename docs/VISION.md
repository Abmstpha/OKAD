# Vision

OKAD should become the default way an agent **explains a system to a human** — and to the next agent.

Imagine:

- Every PR comment can link a journey that changed  
- New hires open `story.html` before they open `src/`  
- Agents refuse to explode file counts because the map shows the seams  

We are not building another RAG index. We are building **shared narrative memory** for software.

If you care about that, open an issue tagged `idea` or start a Discussion.
